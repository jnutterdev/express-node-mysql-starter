# Node.js Rest APIs with Express & MySQL example

A barebones project to bootstrap MySQL, Express.js, and Node.js. Features include:  

- LESS for styling
- Basic user management system 
    - Located under    
- Login/Signup page
- File upload
- Nodemon for hot reloading

## Project setup
```
npm install
```

### Run
```
nodemon server.js
```
